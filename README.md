# UrbanEscape

UrbanEscape is a full-stack web application where users can explore and add listings of beautiful places like hotels, villas, apartments, and more — based on price per night.

Built with Node.js, Express.js, MongoDB, and EJS templates.

---

## Features

- User Authentication & Authorization (Passport.js)
- Add, Edit, and Delete Listings
- Review System (users can leave feedback)
- Map Integration using Mapbox (with Geocoding)
- Filtering and Searching by:
  - Category
  - Location
  - Country
  - Price range
- Image Uploading to Cloudinary
- Database Storage on MongoDB Atlas
- Responsive Design for all devices (optimized for screens < 768px)
- Secure Password Hashing and Salting
- Clean MVC Architecture
- Upcoming Feature: Admin Panel for enhanced UX

---

## Tech Stack

- *Backend:* Node.js, Express.js
- *Frontend:* EJS (Embedded JavaScript Templates)
- *Database:* MongoDB Atlas
- *Authentication:* Passport.js
- *Image Hosting:* Cloudinary
- *Maps & Geocoding:* Mapbox
- *Others:* Mongoose, Method-override, Express-session, Connect-flash
